# 🌍 Global Airbnb Performance Dashboard

An interactive Power BI dashboard analyzing Airbnb listings, pricing trends, market share, growth lifecycle, and city-level performance metrics across major global cities.

This project transforms raw Airbnb listing data into actionable business insights using data modeling, DAX measures, and interactive visual storytelling.

---

## 📊 Project Overview

The Global Airbnb Performance Dashboard provides:

- 📈 Growth lifecycle analysis (Introduction → Growth → Maturity → Decline → Reinvention → COVID-19 impact)
- 🏙 Market share breakdown by city
- ⭐ City-level ratings analysis (Accuracy, Cleanliness, Communication, Location, Value)
- 💰 Average price comparison by property type
- 📦 KPI tracking for listings, hosts, property types, reviews, and cities

---

## 🧠 Business Questions Answered

- How has Airbnb grown over time?
- When did Airbnb reach its peak growth?
- How did COVID-19 impact listings?
- Which cities dominate market share?
- Which cities have the highest customer ratings?
- How do property types compare in pricing?
- Are hotel rooms priced significantly higher than Airbnb listings?

---

## 📌 Key Insights

- 📍 Paris, New York, and Sydney account for nearly half of total listings.
- 📊 Airbnb experienced rapid growth from 2011–2015, peaking in 2015.
- 📉 A decline began in 2016 before stabilizing and reinvention pre-COVID.
- 🦠 COVID-19 caused a sharp drop in listings.
- 🏆 Mexico City and Rio de Janeiro show the highest overall ratings.
- 🧹 Cleanliness and Value tend to be the lowest scoring metrics.
- 💵 Hotel rooms are significantly more expensive compared to entire places.
